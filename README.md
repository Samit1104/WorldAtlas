# WorldAtlas

A small React application that displays country information and details using local JSON data. Built with Vite and React.

## Features

- Country list with search and filter
- Country detail view with key information
- Responsive layout
- Simple, minimal UI components

## Tech Stack

- React (JSX)
- Vite
- Plain CSS

## Project Structure

- `src/` - application source
	- `api/` - local JSON data and API utilities
	- `Components/` - UI components and pages
	- `assets/` - images and static assets
	- `main.jsx` - app entry

## Setup

1. Install dependencies

```bash
npm install
```

2. Run development server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
```

4. Preview production build

```bash
npm run preview
```

## Notes

- Country data is stored in `src/api/countryData.json` for offline/demo use.
- Adjust styles in `src/App.css` and component-level CSS as needed.
